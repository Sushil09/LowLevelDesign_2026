## Tic Tac Toe Game
### Overview
This is a simple implementation of the classic Tic Tac Toe game. The game is played on a nxn grid, where two players take turns marking the spaces with their respective symbols (X and O). The player who succeeds in placing three of their marks in a horizontal, vertical, or diagonal row wins the game. If all nine spaces are filled without a winner, the game is considered a draw.
### Features
- Two-player mode: Players can take turns to play the game.
- Win detection: The game checks for a winner after each move.
- Draw detection: The game checks for a draw if all spaces are filled without a winner.
### Usage
1. Compile the Java code and run the main method in the `TicTacToe` class.
2. Follow the prompts to enter the grid size and player names.
3. Players will take turns entering their moves by specifying the row and column numbers.
4. The game will display the current state of the board after each move and announce the winner or if the game ends in a draw.
